## 📌 Projetos

**[Projeto_Banco_De_Dados](https://github.com/LucasHiratuca/Projeto_Banco_De_Dados)**
Sistema integrado de banco de dados de uma academia integrado com Frontend simples — modelagem com MER e DER, usando SQLite + Conceitos de POO, e Streamlit para o Frontend.
 
**[SQLServer-Docker-AWS](https://github.com/LucasHiratuca/SQLServer-Docker_AWS)**
Pipeline de engenharia de dados de ponta a ponta para um sistema de academia. Banco transacional em SQL Server com Docker, geração de dados sintéticos com Faker, pipeline ETL completo e Data Warehouse dimensional (Star Schema) com `Dim_Aluno`, `Dim_Produto`, `Dim_Data` e `Fato_Venda`. Inclui duas versões do ETL: `pyodbc` + `pandas` e **Apache Spark + Spark SQL**.
 
**[Retail-SQL-Analytics](https://github.com/LucasHiratuca/Retail-SQL-Analytics)**
Análise de dados de varejo com foco em queries SQL avançadas aplicadas a dois datasets independentes em SQLite. Cobre CTEs, Window Functions (`LAG`, `DENSE_RANK`, receita acumulada), JOINs analíticos e CRUD parametrizado completo. Os datasets foram ingeridos de formas diferentes — um via API do Kaggle, outro por download e conversão com pandas — e explorados com Orange Data Mining antes da análise.
 
**[Template-Docker-Spark](https://github.com/LucasHiratuca/Template-Docker-Spark)**
Template de referência para conectar Apache Spark a SQL Server via JDBC — helpers, UDFs e padrões de ETL documentados.
